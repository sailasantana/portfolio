Project Work

DoggosWelcome

If you knew me you'd say this was my passion project. I believe that pet-friendly cities are happier and healthier overall and should be celebrated!

DoggosWelcome enables users to search for dog-friendly activities within a specified zip code, save favorite "DoggoSpots" to their dashboard and recommend dog-friendly businesses to add to DoggosWelcome's directory.

Built With : React, Nodejs, PostgreSQL, Google Maps API, Google Places API, HTML5, CSS3

Outdoorsy

This was the first web application I built to help me practice implementing responsive, mobile-first design and jQuery functionality.

The purpose of the application is to allow outdoor recreationists to find outdoor activities (trails, national parks, ski resorts etc) across this globe. As an additional feature, Outdoorsy also enables the user to gather weather forecasts on their destination of choice.

Built With : jQuery, HTML5, CSS3, ArcGis API, Weather API
